# Wylie

> 다양한 기술을 활용해 사용자의 환경에 맞는 기능을 제공하는 Progressive Web,App을 만들었습니다.

2024.9 ~

## Overview ⭐️

✅ 동적인 스와이퍼 효과를 메인 페이지에 적용   
  


## 사용 기술 스택 🔧

- **Frontend** : PWA, GSAP, HTML, CSS, JavaScript, SCSS
- **Library** : SwiperJS
- **SCM** : Git, GitHub
- **Design** : Figma
- **Platform**: Web,App
